scriptingtask1 - University Data Centre Process and Resource Management System - Bash

- Features: Displays CPU and Memory usage and lists top procesess among others

scriptingtask2 - University High Performance Computing Job Scheduler - Bash

- Features: Checks for large log files and archives them, checks directory disk usage among others

scriptingtask3 Secure Examination Submission and Access Control System - Python

- Features: Administrative controls and logging system

How to run these scripts
- Open terminal
- Copy this repository using: git clone https://github.com/aligoldstraw/Scripting-Exercise.git
- Use cd Scripting-Exercise to get into the correct folder
- Make each script executable by using chmod +x scriptingtask1.sh e.g.
- Run bash script using ./scriptingtask1.sh command
- Run python script using python3 scriptingtask3.py command
